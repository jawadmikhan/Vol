METHODOLOGY

Backtest Design

How the walk-forward simulation is constructed.

PARAMETER VALUE
Data SourceSynthetic reference data (GARCH-simulated returns, mean-reverting VIX)
Backtest Period504 trading days (Jan 2024 - Dec 2025)
Rebalance FrequencyWeekly (every 5 trading days)
PnL ModelGreeks-based attribution: Vega x dVIX + 0.5 x Gamma x dRV^2 + Theta + Correlation
IV SurfaceBase surface scaled by VIX(t) / VIX(baseline=18)
Correlation Matrix90-day rolling pairwise from GARCH-simulated returns
Regime ClassifierMulti-factor score: VIX level, term slope, VVIX, RV/IV spread
Risk LimitsVega bounds [-$2M, +$4M], $500M notional cap, 15% drawdown stop
Transaction Costs5 bps per rebalance (vol targeting strategy)
