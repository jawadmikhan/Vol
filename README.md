# Systematic Volatility Portfolio Process

Interactive workflow dashboard for a dispersion-centric multi-strategy volatility portfolio.

**Live site:** [jawadmikhan.github.io/Vol](https://jawadmikhan.github.io/Vol/)

## Overview

44-step, 7-stage portfolio management workflow covering:

- Signal construction & model calibration
- Collaboration & peer review
- Thesis development & portfolio design
- Decision, approval & execution
- Monitoring & rebalancing

Built as a single self-contained HTML file — no dependencies, no build step.
