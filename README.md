# Systematic Volatility Portfolio Process

Interactive workflow dashboard for a five-leg systematic volatility portfolio.

**Live site:** [jawadmikhan.github.io/Vol](https://jawadmikhan.github.io/Vol)

## Overview

48-step, 7-stage portfolio management workflow covering:

- **Dispersion (Core)** — Index vs constituent volatility mispricing
- **Volatility Harvesting** — Systematic short volatility premium capture
- **Directional Long/Short** — Macro-regime driven volatility views
- **Dynamic Volatility Targeting** — Inverse-volatility leverage scaling
- **Option Overlays** — Protective puts, collars, tail-risk hedges

## Workflow Stages

| Stage | Name | Steps |
|-------|------|-------|
| 1 | Idea Generation & Sourcing | 7 |
| 2 | Research Aggregation | 6 |
| 3 | Collaboration & Peer Review | 5 |
| 4 | Thesis Development | 11 |
| 5 | Decision, Approval, & Portfolio Construction | 7 |
| 6 | Monitoring & Review | 6 |
| 7 | Rebalancing & Liquidity | 6 |

## Blocker Distribution

- **No Blocker (Green):** 34 steps — 71%
- **Soft Blocker (Yellow):** 10 steps — 21%
- **Hard Blocker (Red):** 4 steps — 8%

## Features

- Tabbed navigation across all 7 stages
- Mermaid flowcharts with actor-prefixed nodes on every tab
- Multi-line node labels with context details
- 5 decision gates with feedback loops
- 4 task streams identified for prompt creation
- Full blocker categorization with color coding
- Reference tables, color legend, and strategy leg mapping

Built as a single self-contained HTML file — no dependencies, no build step.
